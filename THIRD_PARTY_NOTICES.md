# 第三方软件声明

清墨包含或在构建时使用以下开源软件。各组件仍遵循其自身许可证；完整许可证文本可在对应包、项目仓库和 `node_modules` 中查阅。

| 组件 | 用途 | 许可证 |
| --- | --- | --- |
| DOMPurify | HTML 清洗 | MPL-2.0 OR Apache-2.0 |
| highlight.js | 代码语法高亮 | BSD-3-Clause |
| marked | Markdown 解析 | MIT |
| iconv-lite | 文本编码转换 | MIT |
| Electron | 桌面运行时 | MIT |
| @electron/packager | Windows 应用打包 | BSD-2-Clause |
| esbuild | 前端构建 | MIT |
| sharp | 图标渲染 | Apache-2.0 |
| Inno Setup | Windows 安装器 | 自定义免费软件许可证 |
| Inno Setup Chinese Simplified Translation | 安装器中文翻译 | MIT |

清墨未包含、加载或执行 Typora 的代码、资源、ASAR、V8 字节码或专有桥接实现。
