# 隐私说明

清墨不包含账号系统、遥测、广告、云端转换、自动同步或协作服务。

## 本地数据

- 打开的文档、转换来源、转换结果和临时文件在本机处理。
- 未保存内容会以明文恢复草稿保存在当前 Windows 用户的应用数据目录中。
- 恢复草稿会在成功保存或用户明确放弃后清理。
- 文档转换使用独立临时目录，成功后原子移动结果；失败或取消时清理临时内容。
- OCR 识别在本机执行，不向 OCR 服务上传图片或文字。

## 可能发生的联网

只有以下用户可见或用户主动允许的功能会联网：

1. 用户在转换中心确认下载 OCR 语言包时，从 jsDelivr 获取固定版本的 chi_sim 或 eng 数据；清墨会校验固定大小和 SHA-256。
2. 文档包含远程图片且用户明确允许加载时，通过受限 HTTPS 图片通道请求该图片。
3. 用户主动打开项目主页、支持页面或外部组件官方下载页时，由默认浏览器访问对应网站。

清墨不会把当前文档、文件名、转换内容或 OCR 结果附加到上述请求。

## 外部本机程序

用户已经安装的 Microsoft Office、LibreOffice 或 Pandoc 可能被清墨用于本地转换。清墨不会静默安装它们。其自身的隐私行为和更新策略由对应软件供应方负责。

## 卸载

正常卸载时可选择是否一并删除清墨的本地设置和恢复草稿。转换输出和用户文档不会被卸载程序删除。
