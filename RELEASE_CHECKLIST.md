# 清墨 0.3.0 发布清单

## 功能与数据安全

- [x] 新建、打开、保存、另存为及真实 Markdown 文件流程
- [x] 撤销与重做（工具栏和快捷键）
- [x] 同目录临时文件、刷盘与原子替换保存
- [x] 外部文件修改冲突检测和用户确认
- [x] 未命名及已有文档的崩溃恢复草稿
- [x] UTF-8 BOM、UTF-16 LE/BE、GB18030 检测与保存
- [x] 原编码、BOM 和 LF/CRLF/CR 换行保留
- [x] 远程图片默认阻止，用户点击后才加载
- [x] 亮色和暗色语法高亮配色
- [x] 禁止新窗口和页面导航，保留沙箱及 HTML 清洗

## 构建与安装

- [x] 版本统一取自 `package.json`（0.3.0）
- [x] Electron 本地 ZIP 缓存可选，干净克隆可联网构建
- [x] 图标完全使用 Node.js 生成，不依赖 Python/Pillow
- [x] 生产包排除测试、源码、构建脚本和安装脚本
- [x] 当前用户中文安装器、可选桌面快捷方式与“打开方式”
- [x] 安装器文件名：`清墨-0.3.0-安装程序.exe`
- [x] 安装器 SHA-256 校验值随 GitHub Release 生成
- [ ] Windows 代码签名（需要开发者提供有效代码签名证书）
