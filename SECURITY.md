# 安全政策

当前受支持版本为最新 GitHub Release。请不要在公开 Issue 中披露尚未修复的敏感漏洞，请通过 [GitHub 私密安全公告](https://github.com/ZuoXing-0504/MarkDown-Free/security/advisories/new) 报告。

报告请包含受影响版本、复现步骤、影响、示例文件和建议修复方式。请避免附带真实机密文档；可以使用最小化测试文件。维护者会尽快确认并协调披露时间。

## 转换安全边界

- 文档转换在本机完成，不上传来源文件或结果。
- 外部输入、输出路径必须由用户通过原生文件对话框授权。
- DOCX、PPTX、PDF、图片和 OCR 均设置大小、页数、像素及超时限制。
- OOXML 解压会限制总大小和单项大小，防止压缩炸弹。
- SVG 会拒绝脚本、事件、外部资源、DOCTYPE 和 CSS 外链，并安全栅格化。
- 远程图片默认关闭；开启后只允许受限的公开 HTTPS 图片资源。
- Office 自动化使用清墨独立创建的只读实例，禁用宏、链接更新和交互提示。
- 转换失败或取消时不会修改来源文件，并会清理临时文件。

恶意文档解析器始终可能出现新的漏洞。请保持清墨和本机 Office/LibreOffice/Pandoc 为受支持版本，不要打开来源不明的文档。

## 发布完整性

清墨目前没有商业代码签名。请只从本项目 GitHub Release 下载 Windows 安装器，并核对发布页提供的 SHA-256。
